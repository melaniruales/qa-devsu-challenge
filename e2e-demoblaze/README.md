\# QA Devsu Challenge - E2E Automation



\## Proyecto

Automatización E2E del flujo de compra en Demoblaze utilizando Playwright y JavaScript.



\## Tecnologías

\- Playwright

\- JavaScript

\- Node.js



\## Escenario automatizado

El flujo automatizado incluye:



1\. Agregar dos productos al carrito

2\. Visualizar el carrito

3\. Completar formulario de compra

4\. Finalizar compra

5\. Validar mensaje:

&#x20;  "Thank you for your purchase!"



\## Arquitectura

El proyecto utiliza:

\- Page Object Model (POM)

\- Separación por páginas

\- Captura automática de screenshots

\- Reporte HTML de Playwright
